# EditorJs + Nuxt 4 Playground

Промежуточный `Nuxt 4` playground-проект для отработки архитектуры визуального редактора контента на базе `Editor.js`, `Vue 3` и `TypeScript`.

Цель репозитория:

- собрать устойчивую frontend-основу editor-layer и renderer-layer;
- работать со статическим JSON и `localStorage` без бэкенда на раннем этапе;
- подготовить проект к дальнейшему росту: custom blocks, tunes, nested editors и GitHub Pages deployment.

Детальные требования и порядок реализации зафиксированы в [SPEC.md](./SPEC.md).
